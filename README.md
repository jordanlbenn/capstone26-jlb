# EDIS – Encryption Decision and Implementation System

## Overview
The **Encryption Decision and Implementation System (EDIS)** is a cybersecurity-focused application designed to simplify the process of selecting and applying appropriate encryption methods. Many users lack the expertise to determine which cryptographic technique best fits their needs. EDIS addresses this gap by combining a **decision-making engine** with **practical encryption functionality**.

The system guides users through encryption choices and allows them to securely encrypt and decrypt files using recommended methods.

---

## Objectives
- Simplify encryption method selection for non-experts  
- Demonstrate real-world encryption workflows  
- Provide hands-on implementation of symmetric, asymmetric, and hybrid encryption  
- Highlight the importance of data protection and secure communication  

---

## Features
- **Encryption Decision Engine**  
  Recommends encryption methods based on user input (e.g., data sensitivity, use case)

- **File Encryption & Decryption**  
  Securely encrypt and decrypt files using selected methods  

- **Hybrid Encryption Support**  
  Combines symmetric and asymmetric encryption for improved efficiency and security  

- **Security Demonstration**  
  Shows that encrypted data cannot be accessed without the correct key  

---

## How It Works
1. User inputs information about their data and use case  
2. The decision engine recommends an encryption method  
3. The system applies the selected encryption technique  
4. The user can decrypt the file using the correct key  
5. Unauthorized access attempts fail without the key

---

## Security Considerations
- Encryption protects data confidentiality but depends heavily on secure key management
- EDIS demonstrates encryption concepts but uses simplified key handling
- Not intended for production-level deployment without additional security enhancements

## Limitations
- Limited set of supported encryption algorithms
- No advanced key management system
- Basic user interface
- Not optimized for large-scale or high-performance use

## Future Improvements
- Advanced encryption algorithm support
- Secure key management (generation, storage, exchange)
- Improved user interface and user experience
- Cloud integration for real-time encryption
- AI-driven encryption recommendations
- Performance optimization for large datasets
